# Arduino-FM-radio

Arduino-based FM radio tuner with rotary encoder tuning, I2C LCD display. Features a custom needle-on-scale UI and station name lookup for Dalmatian/Croatian radio stations.

The first version was school project and my first project. The code is bad, but it worked :).

The second version in branch Version2 is an improved design and it only lacks the buttons for presets. The new design incorporates an rtc clock for tracking time and eeprom to remmber the last frequency.
